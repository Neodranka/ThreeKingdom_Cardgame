# 三国杀卡牌游戏

一个基于 Unity 开发的三国杀风格卡牌游戏

## 项目信息

- **Unity 版本**: 2022.3 LTS
- **开发语言**: C#
- **目标平台**: PC / WebGL / Android


## 开发环境设置

### 克隆项目
```bash
git clone https://github.com/Neodranka/ThreeKingdom_Cardgame.git
cd YOUR_REPO_NAME
git lfs pull  # 如果使用了 LFS
```

### 打开项目
1. 打开 Unity Hub
2. 点击 "Add" 或 "Open"
3. 选择克隆的项目文件夹
4. 等待 Unity 导入资源

### 开发规范
- 使用 Feature 分支开发
- 提交前测试功能
- 遵循代码规范（见 CONTRIBUTING.md）
